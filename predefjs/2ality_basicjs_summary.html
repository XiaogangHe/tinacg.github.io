Syntax

There are two syntactic categories: statements and expressions.

Statements are like instructions, they "do things." 

var animal;

Expressions produce values 

console.log(2 * 9);
// 18

If-then-else may be written as both a statement or expression.

Variables

Variables must be declared before they are used. An assignment may be done with the declaration. An existing variable can be re-assigned a new value. 

var n = 3;
console.log(n);
// 3

n = 6;
console.log(n);
// 6
